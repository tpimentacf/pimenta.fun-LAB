Lab Discovery & Implementation Guide

Every implementation in the Pimenta.fun Cloudflare lab, in one place — what each one demonstrates and how it is wired up. Use the filter to jump to a feature.

Architecture. The lab runs on a single origin (Ubuntu + Apache) behind Cloudflare across three hostnames — www (this site / tests), shop (OWASP Juice Shop storefront) and api (Juice Shop API). Most pages are static origin files; the Cloudflare behaviour they exercise is configured in the dashboard, with a few Workers doing edge logic. See the deployment guide.
