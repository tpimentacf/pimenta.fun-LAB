Zaraz

Cloudflare Zaraz

Load and manage third-party tools (analytics, pixels, chat) from Cloudflare's edge instead of the browser — faster pages, fewer scripts, and a built-in consent manager. This page talks to the live window.zaraz API.

Configure first: Dashboard → Zaraz. Add a tool (e.g. a custom HTML/ analytics tool) and an action/trigger. Cloudflare injects zaraz.js automatically on the zone — this page detects it below.
Use at your own risk. This lab was built by TPimenta LAB for hands-on testing. Always consult the official Cloudflare documentation for accurate, up-to-date guidance.

Web API reference

CallPurpose
zaraz.track(name, props)Send a custom event to subscribed tools.
zaraz.set(key, value)Set a variable for the page/session, available to tools.
zaraz.ecommerce(name, data)Send standardized e-commerce events (e.g. "Order Completed").
zaraz.consent.get(id)Read consent for a purpose ID.
zaraz.consent.setAll(bool)Grant or withdraw all purposes at once.
zaraz.showConsentModal()Open the consent modal programmatically.
