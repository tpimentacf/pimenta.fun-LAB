CF Echo

Cloudflare request.cf Echo

A Cloudflare Worker reflects your request back exactly as the edge sees it — including the request.cf object and the headers Cloudflare injects that client-side JavaScript can never read.

Companion to the Edge Inspector. That page reads what the browser can see (/cdn-cgi/trace + response headers). This page asks a Worker server-side, so you also get CF-Connecting-IP, CF-IPCountry, ASN, TLS cipher, bot score, and the rest of request.cf.
Use at your own risk. This lab was built by TPimenta LAB for hands-on testing. Always consult the official Cloudflare documentation for accurate, up-to-date guidance.
